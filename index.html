1

Buy a GL.iNet Mango router

Search Amazon India for GL.iNet GL-MT300N-V2. Cost: ₹1,800–2,500. It comes with OpenWrt pre-installed — no technical setup needed.

→ Search on Amazon India
2

Connect it to your main router

Use a regular Ethernet cable:
Main router's yellow LAN port → GL.iNet's blue WAN port
Plug GL.iNet into power. Wait 30 seconds for it to boot.

3

Log into GL.iNet dashboard

Connect your laptop to the GL.iNet WiFi (sticker on bottom shows default password). Open browser → go to 192.168.8.1. Create a new admin password when asked.

4

Set your guest WiFi name

In GL.iNet dashboard → Wireless → Set WiFi name (SSID) to: MyPass-Guest
Set any password for now. MyPass will rotate it automatically.

5

Install Node-RED on the router

In GL.iNet dashboard → Applications → Plug-ins → search node-red → click Install.
Takes 2–3 minutes. After install, open: 192.168.8.1:1880

6

Import the MyPass automation flow

1. Go to 192.168.8.1:1880
2. Click ☰ menu (top-right) → Import
3. Click the button below to copy the flow, paste it in Node-RED
4. Click Import then the red Deploy button

7

Link router to MyPass — final step!

Enter your GL.iNet router's IP (usually 192.168.8.1) and click Link. This tells your cloud where to send commands.
